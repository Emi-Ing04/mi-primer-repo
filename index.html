Dispositivo Lúdico REBO

Proyecto de titulación para Ingeniería Mecánica.

¿Qué es este proyecto?

Este es el panel de control web para un dispositivo electrónico con función lúdica, inspirado en el juego clásico de "Simón Dice". El objetivo principal es ayudar a los usuarios a memorizar notas musicales a través de la interacción física.

El diseño tiene un fuerte enfoque inclusivo: está pensado principalmente para personas con discapacidad visual, garantizando al mismo tiempo que las personas normovisuales puedan utilizarlo y disfrutarlo por igual.

Estado Actual

Actualmente el desarrollo de hardware se encuentra en una etapa avanzada. Los logros hasta el momento incluyen:

  • Diseño y manufactura de las piezas mecánicas.
  • Ensamblaje de dos prototipos funcionales utilizando tecnología Arduino.
  • Pruebas iniciales de lógica de programación y respuesta de componentes electrónicos.

Próximos Pasos

Las siguientes fases del proyecto involucran el uso de programación web y recolección de información para aplicar estrategias de análisis de datos. Esto permitirá evaluar la usabilidad del producto final.
